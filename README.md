## Usage

```bash
git clone https://github.com/Teraskull/magpi-downloader

cd magpi-downloader

sudo chmod +x magpi-issue-downloader.sh

./magpi-issue-downloader.sh
```

All issues will be downloaded under the `issues` directory.
